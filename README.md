# riscv-wch-elf-gcc

This is the WCH fuxking modified RISC-V GCC toolchain repo.

Current version is `riscv-wch-elf-gcc15`

If you have a fuxking WCH MCU project, you can use this repo to build your own Github CI process.

## Using steps

1. clone this repo

2. use `tar` to unpress the `MRS_Toolchain_Linux_x64_V240.tar.xz` pack

3. add environment variables

4. use your build system
